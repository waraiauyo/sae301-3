# Site de la SAE3.01 / 3.03 v1

Création d'un site qui permet aux étudiants, de trouver un master prêt de chez eux, facilement.

## Technologies

**Client** : React, Tailwind, Rechart.

**Serveur** : Node, NextJS, ( +Prisma pour la version finale ).

## Installation local

Cloner le répertoire ( actuellement la v1 ).

```bash
  git clone https://github.com/waraiauyo/sae301-3.git
```

Ce déplacer à la racine du projet.

```bash
  cd sae301-3
```

Installer les dépendances.

```bash
  npm install
```

Démarrer l'application en local.

```bash
  npm run dev
```

Et pour terminer, rendez-vous sur [http://localhost:3000](http://localhost:3000 "http://localhost:3000").
